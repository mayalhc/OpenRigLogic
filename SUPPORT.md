## Support

To discuss OpenRigLogic, or to ask questions please use the [MetaHuman category of the Unreal Engine forums](https://forums.unrealengine.com/categories?tag=metahuman).

If you would like to report an issue, first check the [list of issues](https://github.com/epicgames/OpenRigLogic/issues) to see if it has already been reported.  For new issues, use the **Bug Report template**. Please include:

- A clear, descriptive title.  
- Steps to reproduce the issue.  
- Expected versus actual behaviour.  
- Your environment: operating system, compiler version, Unreal Engine version, OpenRigLogic branch or tag.  
- Logs, screenshots, or a minimal reproduction project if possible.

Ideas for a new feature can be reported in a similar way, using the **Feature Proposal template**.  

We aim to provide an initial response to issues within **four weeks**, though response times may vary.
